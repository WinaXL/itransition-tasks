# Задание 4 — Surname Levenshtein Outlier

Найти фамилию, которая **максимально отличается** от всех остальных в датасете `in.txt`.

## Метрика

Среднее расстояние Левенштейна от каждой фамилии до стратифицированной выборки из 2 000 имён. Фамилия с наибольшим средним значением — статистический выброс.

## Запуск

```bash
node task4-surname-outlier/surname-levenshtein-outlier.js path/to/in.txt
```

Если путь не указан, скрипт ищет `in.txt` в своей папке.

## Результат

**SCHIMMELPFENNIG** — avg Levenshtein distance ≈ 12.87 (15 символов, редкие немецкие диграфы).
