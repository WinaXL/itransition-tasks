# Задание 3 — LCM Web Server (Express)

HTTP-сервер на **Node.js + Express**, который по двум натуральным числам вычисляет их **наименьшее общее кратное (LCM)** и возвращает результат в виде plain text.

## Быстрый старт

```bash
cd task3
npm install
npm start
```

Сервер запустится на `http://localhost:3000`.

## API

| | |
|---|---|
| **Метод** | `GET` |
| **Путь** | `/:emailSlug` |
| **Query** | `x`, `y` — натуральные числа (целые > 0) |
| **Ответ** | plain text: число или строка `NaN` |

`emailSlug` — произвольный сегмент пути (например, email, в котором все символы кроме букв и цифр заменены на `_`).

### Примеры

| Запрос | Ответ | Пояснение |
|--------|-------|-----------|
| `GET /john_doe_example_com?x=15&y=20` | `60` | LCM(15, 20) = 60 |
| `GET /user?x=12&y=18` | `36` | LCM(12, 18) = 36 |
| `GET /user?x=0&y=5` | `NaN` | 0 не является натуральным |
| `GET /user?x=-3&y=5` | `NaN` | отрицательные числа недопустимы |
| `GET /user?x=1.5&y=5` | `NaN` | дробные числа недопустимы |
| `GET /user?x=4` | `NaN` | отсутствует параметр `y` |

```bash
curl "http://localhost:3000/john_doe_example_com?x=15&y=20"
# 60
```

## Алгоритм

1. **Валидация** — `x` и `y` должны быть натуральными числами. Иначе → `NaN`.
2. **GCD** — наибольший общий делитель через [алгоритм Евклида](https://en.wikipedia.org/wiki/Euclidean_algorithm).
3. **LCM** — формула `lcm(a, b) = (a / gcd(a, b)) * b` (деление выполняется до умножения, чтобы избежать переполнения).
